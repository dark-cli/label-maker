🏷️ Label Maker QR Redirect Service

This service redirects short QR codes to the full label descriptions hosted on GitHub.

Usage

Short URL:
https://dark-cli.github.io/label-maker/?code=AC-OUT-DIST-F
Redirects to:
https://raw.githubusercontent.com/dark-cli/label-maker/master/qr_codes/AC-OUT-DIST-F.txt

Examples

  • AC-CHG-IN - Power In
  • ETH-ROOM - Ethernet to Ali's Room
  • HOME-SARA - Sara's Port

Scan any QR code generated by the Label Maker app to view label descriptions.
